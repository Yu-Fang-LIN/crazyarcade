## 操作方式:

#### 方向鍵、wasd:前後左右

#### shift:放炸彈

### 玩家 1:

#### z:開槍

#### x:放地雷

### 玩家 2:

#### cril:開槍

#### alt:放地雷

### 玩家:綠色、藍色

### 木頭(會被炸掉):橘色

### 石頭(不會被炸掉):灰色

### 炸彈:紅色

### 地雷:黃色

### 威力藥水:紫色

### 加速藥水: 粉紅色
